Linux Grafikus Felület (GUI)

A Linux grafikus felülete lehetővé teszi a felhasználók számára, hogy parancssor helyett vizuális módon kezeljék a rendszert.

Egyszerű, letisztult és könnyen bemutatható környezet.

Flatpak Parancsok

  • install – Alkalmazás telepítése
  • update – Telepített alkalmazás frissítése
  • uninstall – Alkalmazás eltávolítása
  • list – Telepített alkalmazások listázása
  • info – Információ megjelenítése
  • config – Flatpak beállítása
  • repair – Hibás telepítések javítása
  • create-usb – Telepítés áthelyezése USB-re
  • search – Távoli alkalmazások keresése
  • run – Alkalmazás futtatása

Minimum Rendszerkövetelmények

  • 1 GHz processzor (pl. Intel Celeron)
  • 1.5 GB RAM
  • 10 GB szabad merevlemez terület
  • CD/DVD meghajtó vagy USB port
  • Internet kapcsolat ajánlott
